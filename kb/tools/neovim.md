# NeoVim

# Setup
## nvchad
https://nvchad.com/docs/quickstart/install

- PowerShell
~~~bash
git clone https://github.com/NvChad/NvChad $ENV:USERPROFILE\AppData\Local\nvim --depth 1 
~~~

## MinGW (GCC)
~~~bash
choco install -y mingw
~~~

## Install help parser
~~~
:TSUpdate vimdoc
~~~
